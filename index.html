asteriskcdr

Simple asterisk call data records in ruby, sinatra and angularJs VERSION 1.0

Initial setup:

Copy and change yml files:

cp config/config.sample.yml config/config.yml

cp config/database.sample.yml config/database.yml

Optional change config/deploy.rb and deploy/production.rb to use it with your own server.

Run bundle to install required gems

To run locally with thin server:

gem install thin

thin start
